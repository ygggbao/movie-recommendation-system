## 💻 系统架构

### 后端API (Flask)
- **健康检查：** `GET /api/health`
- **获取推荐：** `GET /api/recommendations/{user_id}?algorithm=itemcf&num=10`
- **电影信息：** `GET /api/movies/{movie_id}`
- **用户评分：** `GET /api/users/{user_id}/ratings`

### 前端界面 (HTML/JS)
- **响应式设计：** 适配不同屏幕尺寸
- **实时交互：** 异步API调用
- **美观界面：** 渐变背景、卡片布局

### 神经网络架构
```
用户ID → Embedding(32) ┐
                         ├─ 拼接 → MLP(64,32) → 输出
电影ID → Embedding(32) ┘
```
**参数量：** 98,401个  
**训练轮数：** 10 epochs  
**最终损失：** 0.7501
